addEventListener(eventType, eventHandlerFunction) 


const inputElements = document.getElementsByTagName("#js-logo");
inputElements.addEventListener("click", onEventHandler);

function onEventHandler(e) {
document.body.style.backgroundColor="green"
}
Step 1
Create a reference variable that contains the DOM Element you would like to attach an event too.
Step 2
Add the addEventListener() method to the reference variable
Step 3
Add the event type as the first argument to the addEventListener() method
Step 4
Add a custom event handler function to the addEventListener() method. Name this function so that the action of the event is clear.
Step 5
Create the custome event handler function for the addEventListener() method. Note that you should add the letter e between the brackets. This is short for event. In the body of the function, between the two curly braces change the background color of the body element.
